📁 Setup Google Drive Folder

PDF laporan PM akan tersimpan otomatis ke Google Drive setelah submit laporan.

Cara mendapatkan Folder ID:
1. Buka Google Drive → pilih/buat folder tujuan
2. Klik folder → Copy URL dari browser
3. Ambil ID dari: drive.google.com/drive/folders/[ID INI]
⚠️ Pastikan folder ini sudah dishare dengan akun Google yang menjalankan Apps Script, atau folder ada di My Drive akun tersebut.
